

# SheFit 

### Project Description
**SheFit** is an online platform that provides information about women-only gyms in Riyadh, including subscription prices, user ratings, and personal trainers. The platform helps users find suitable gyms and connect with trainers, whether they belong to a gym or work independently as personal trainers.

### Features
- Display a list of available women-only gyms in Riyadh.
- Show subscription prices for each gym.
- Gym rating system based on trainee reviews.
- Display personal trainers and indicate whether they belong to a gym or are independent.
- Search and filter by location, price, gym rating, or trainer type.
- A detailed page for each gym with services and schedules.
- A profile page for each personal trainer with contact info, experience, and number of trainees.

### User Stories

As an administrator
1- As an admin, I want to be able to delete any gym or trainer account.
2- As an admin, I want to manage and moderate comments for all gyms and trainers.
3- As an admin, I want to respond to user inquiries through email so that I can provide support.

As a trainee
1- As a trainee, I want to view all gyms and all personal trainers so that I can explore my options.
2- As a trainee, I want to see a detailed page for each gym so that I can get complete information about its services, prices, and schedules.
3- As a trainee, I want to rate a gym if I have subscribed to it or rate a personal trainer if I have trained with them.
4- As a trainee, I want to view the personal profile page of a trainer to learn about their experience, services, and contact information.
5- As a trainee, I want to write comments on a gym’s page based on my inquiries, feedback, or visits.
6- As a trainee, I want to write comments on a personal trainer’s profile page to provide feedback or share my experience.

As a gym admin
1- As a gym, I want to create an account and add my gym’s information, prices, and services so that I can attract trainees.
2- As a gym, I want to respond to users’ questions and inquiries on my gym page so that I can provide support and clarify any information.

As a coach
1- As a coach, I want to choose whether I am associated with a gym or work independently so that users can know my training type.
2- As a coach, I want to have my own personal profile that displays my information, experience, and subscription price so that trainees can learn about my services.
3- As a coach, I want to respond to users’ comments on my profile so that I can answer their questions and provide support.
